Вариант 48

Разработать калькулятор для реализации операций с комплексным числом, зная в какой форме задавать комплексное число: показательной или тригонометрической. Калькулятор должен вычислять:

  1. действительную часть числа;
  2. модуль числа;
  3. мнимую часть числа;
  4. представление в остальных формах.

Калькулятор должен включать:

  • поле со списком для выбора формы представления комплексного числа (показательной или тригонометрической) и кнопку Показать;
  • поля для ввода компонентов комплексного числа;
  • группу переключателей (checkbox) для выбора одной или нескольких операций;
  • кнопки Вычислить и Очистить.

В начальный момент времени на странице должна отобразиться форма представления комплексного числа, выбранная в поле со списком, поля для ввода комплексных чисел, а также операции, перечисленные в группе переключателей.

После выбора пользователем новой формы представления комплексного числа и клику на кнопку Показать, на странице при необходимости должны измениться подписи полей для ввода данных.

После ввода пользователем входных данных, выбора операций и клику по кнопке Вычислить, на той же странице вывести результаты расчетов. Результаты выводить той же форме, в которой задавалось комплексное число. Вся введенная пользователем информация в формах после обновления страницы должна сохраняться.

При клике на кнопку Очистить – очистить поля для ввода комплексного числа.

Примечание. Для реализации использовать Python, JINJA, при необходимости другие библиотеки Python. JavaScript НЕ ИСПОЛЬЗОВАТЬ.
